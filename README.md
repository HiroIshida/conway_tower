## conway_tower
Fast enough random 3d limestone-cave-like shape generation using 2d Conway's Game of Life. (inspired by https://www.youtube.com/watch?v=dfzdSl9zYog)

Installation by
```python
pip3 install scikit-build
pip3 install -e . -v
```
and try `example.py` to get:
